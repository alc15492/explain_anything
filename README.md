# ExplainAnything

This project explores how to use AI to produce high-quality educational content at scale, by generating LLM-based content and having users provide feedback.

Features include:
- LLM-based content generation
- Vector search for content discovery
- Page to display articles generated
- Specialized links within article that generate related articles
- Manual user edits within Lexical editor
- AI-driven suggestions triggered from AI Editor module
